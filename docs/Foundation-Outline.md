# Purpose

The Foundation Statement is the constitutional document of the TLM project. It establishes the canonical concepts, terminology, assumptions, design principles, evaluation framework, and long-term research agenda upon which future research notes and mechanism proposals are built.

Rather than proposing mechanisms, it defines the conceptual framework within which mechanisms can later be evaluated.

# Writing Principles

- Concepts before mechanisms.
- Separate economic concepts, representations, and mechanisms.
- State assumptions explicitly.
- Define terminology once and reference it consistently.
- Preserve participant neutrality and protocol neutrality.
- Evaluate future mechanisms using common criteria.
- Build upon, rather than replace, existing Ethereum research.
